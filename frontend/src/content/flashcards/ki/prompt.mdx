    <div slot="question">
        Was ist ein Prompt?
    </div>

    <div slot="answer">
        Ein **Prompt** ist die Eingabe (Text, Bild oder eine Kombination), die ein Nutzer an ein KI-Modell sendet, um eine gewünschte Ausgabe zu erzeugen. Der Prompt steuert, was und wie das Modell antwortet.
        <br/>
        **Bestandteile eines Prompts** können sein:
        - **Instruktion**: Was soll das Modell tun? (z. B. „Fasse zusammen", „Übersetze")
        - **Kontext**: Hintergrundinformationen für die Aufgabe.
        - **Eingabedaten**: Die konkreten Daten, die verarbeitet werden sollen.
        - **Ausgabeformat**: In welcher Form die Antwort erwartet wird.
        <br/>
        **Prompt Engineering** bezeichnet die systematische Gestaltung von Prompts, um die Qualität der Modellantworten zu optimieren.
    </div>
</Flashcard>
