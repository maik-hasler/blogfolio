<Flashcard>
    <div slot="question">
        Welche IT-relevanten Vertragsarten gibt es?
    </div>

    <div slot="answer">
        | Vertragsart | Beschreibung | Beispiel |
        |-------------|-------------|---------|
        | **Kaufvertrag** | Kauf einer Sache gegen Geld (§433 BGB) | Hardware kaufen, Softwarelizenz (einmalig) |
        | **Werkvertrag** | Herstellung eines **bestimmten Ergebnisses** (§631 BGB) | Individualsoftware, Website erstellen |
        | **Dienstvertrag** | Erbringung einer **Tätigkeit** (§611 BGB) | IT-Beratung, Arbeitnehmervertrag |
        | **Mietvertrag** | Überlassung einer Sache auf Zeit (§535 BGB) | Server-Housing, Hardware-Leasing |
        | **Lizenzvertrag** | Nutzungsrecht an Software | SaaS-Abo, Microsoft 365 |
        | **Wartungsvertrag** | Regelmäßige Pflege/Support | Software-Updates, Hardwarewartung |

        <br/>
        **Werkvertrag vs. Dienstvertrag:**

        | | Werkvertrag | Dienstvertrag |
        |---|-----------|-------------|
        | **Geschuldet** | **Erfolg** (funktionierendes Ergebnis) | **Tätigkeit** (Bemühen) |
        | **Abnahme** | Ja, Abnahme durch Auftraggeber | Nein |
        | **Mängelhaftung** | Ja, Gewährleistung | Nein |
        | **Beispiel** | Website mit 5 Seiten erstellen | 10 Stunden IT-Beratung |

        <br/>
        **IHK-Prüfung:** Oft wird gefragt, ob ein Szenario ein Werk- oder Dienstvertrag ist!
    </div>
</Flashcard>
