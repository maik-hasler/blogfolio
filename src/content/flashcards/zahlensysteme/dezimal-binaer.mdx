<Flashcard>
    <div slot="question">
        Wie rechnet man Dezimal in Binär um (und umgekehrt)?
    </div>

    <div slot="answer">
        **Dezimal → Binär (Divisionsmethode):**
        Die Dezimalzahl wird wiederholt **durch 2 geteilt**, die **Reste** ergeben die Binärzahl (von unten nach oben gelesen).

        <br/>
        **Beispiel: 42 → Binär**
        ```
        42 ÷ 2 = 21  Rest 0
        21 ÷ 2 = 10  Rest 1
        10 ÷ 2 = 5   Rest 0
         5 ÷ 2 = 2   Rest 1
         2 ÷ 2 = 1   Rest 0
         1 ÷ 2 = 0   Rest 1
        → von unten lesen: 101010
        ```

        <br/>
        **Binär → Dezimal (Stellenwertverfahren):**
        Jede Stelle wird mit der entsprechenden **Zweierpotenz** multipliziert und aufsummiert.

        <br/>
        **Beispiel: 101010 → Dezimal**
        ```
        1×2⁵ + 0×2⁴ + 1×2³ + 0×2² + 1×2¹ + 0×2⁰
        = 32  +  0   +  8   +  0   +  2   +  0
        = 42
        ```

        <br/>
        **Zweierpotenzen merken:**
        2⁰=1, 2¹=2, 2²=4, 2³=8, 2⁴=16, 2⁵=32, 2⁶=64, 2⁷=128, 2⁸=256
    </div>
</Flashcard>
