  <div slot="question">
    Was ist ein Lambda-Ausdruck und was sind seine wesentlichen Merkmale?
  </div>

  <div slot="answer">
    Ein Lambda-Ausdruck, auch **anonyme Funktion**, ist eine Funktionsdefinition, die nicht an einen Bezeichner gebunden ist.
      Sie können als Wert verwendet werden (z. B. als Argument einer Funktion höhrer Ordnung oder Rückgabewert).

      <br/>
      **Wesentliche Merkmale:**
    * **Anonym:** Kein eigener Bezeichner
    * **Kompakte Syntax:** Kurzschreibweise mit bspw. =>
    * **Erstklassiges Objekt:** Kann zugewiesen und übergeben werden
    * **Closure:** Zugriff auf Variablen des umgebenden Scopes

    <br/>

    Beispiel in C#:
    ```csharp
    var list = new List<int> { 1, 2, 3 };
    list.ForEach(x => Console.WriteLine(x));
    ```
  </div>
