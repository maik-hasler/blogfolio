    <div slot="question">
        Was ist der Dunning-Kruger-Effekt?
    </div>

    <div slot="answer">
        Der **Dunning-Kruger-Effekt** ist eine kognitive Verzerrung, bei der Menschen mit **geringer Kompetenz** ihre Fähigkeiten **überschätzen**, während **Experten** ihre Fähigkeiten eher **unterschätzen**.

        <br/>
        **Die typische Kurve:**
        ```
        Selbstvertrauen
        ▲
        │   ╱╲  "Berg der Ahnungslosigkeit"
        │  ╱  ╲
        │ ╱    ╲
        │╱      ╲    "Tal der Verzweiflung"
        │        ╲         ╱─────── "Plateau der
        │         ╲───────╱         Nachhaltigkeit"
        │
        └──────────────────────────► Kompetenz
        ```

        <br/>
        **Phasen:**
        1. **Anfänger:** Weiß nicht, was er nicht weiß → überschätzt sich
        2. **Fortgeschritten:** Erkennt die Komplexität → Selbstvertrauen sinkt
        3. **Erfahren:** Wächst stetig in Kompetenz und realistischer Selbsteinschätzung

        <br/>
        **Relevanz in der IT:**
        - Nach einem Tutorial denkt man, man beherrscht ein Framework
        - In der Praxis merkt man, wie viel man noch nicht weiß
        - Das ist **normal** und gehört zum Lernprozess!

        <br/>
        **Gegenmittel:** Feedback einholen, Code Reviews, Demut, kontinuierliches Lernen.
    </div>
</Flashcard>
