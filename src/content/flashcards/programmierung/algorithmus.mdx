    <div slot="question">
        Was ist ein Algorithmus?
    </div>

    <div slot="answer">
        Ein **Algorithmus** ist eine **endliche, eindeutige Folge von Anweisungen**, die ein bestimmtes Problem löst.

        <br/>
        **Eigenschaften eines Algorithmus:**
        - **Eindeutigkeit:** Jeder Schritt ist klar definiert
        - **Ausführbarkeit:** Jeder Schritt ist tatsächlich durchführbar
        - **Endlichkeit (Finitheit):** Der Algorithmus endet nach einer endlichen Anzahl von Schritten
        - **Determiniertheit:** Bei gleicher Eingabe liefert er immer das gleiche Ergebnis

        <br/>
        **Darstellungsformen:**
        - Pseudocode
        - Programmablaufplan (Flussdiagramm)
        - Struktogramm (Nassi-Shneiderman-Diagramm)
        - Programmcode
    </div>
</Flashcard>
