<Flashcard>
    <div slot="question">
        Was bedeutet das Schlüsselwort static?
    </div>

    <div slot="answer">
        **static** bedeutet, dass ein Member (Attribut oder Methode) zur **Klasse** gehört und nicht zu einer bestimmten **Instanz** (Objekt).

        <br/>
        **Eigenschaften:**
        - Existiert **nur einmal** im Speicher, unabhängig von der Anzahl der Objekte
        - Kann **ohne Instanziierung** über den Klassennamen aufgerufen werden
        - Statische Methoden können **nicht** auf Instanzvariablen zugreifen

        <br/>
        **Beispiel in C#:**
        ```csharp
        class Zaehler
        {
            public static int Anzahl = 0; // gehört zur Klasse

            public Zaehler()
            {
                Anzahl++; // wird bei jeder Instanziierung erhöht
            }
        }

        new Zaehler();
        new Zaehler();
        Console.WriteLine(Zaehler.Anzahl); // 2 (Aufruf über Klasse)
        ```

        <br/>
        **Bekannte Beispiele:** `Math.PI`, `Math.Max()`, `Console.WriteLine()`, `int.Parse()`
    </div>
