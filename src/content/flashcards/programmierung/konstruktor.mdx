    <div slot="question">
        Was ist ein Konstruktor?
    </div>

    <div slot="answer">
        Ein **Konstruktor** ist eine spezielle Methode, die beim **Erstellen eines Objekts** automatisch aufgerufen wird. Er dient dazu, das Objekt zu initialisieren.

        <br/>
        **Merkmale:**
        - Hat den **gleichen Namen** wie die Klasse
        - Hat **keinen Rückgabetyp** (auch nicht `void`)
        - Wird mit `new` aufgerufen
        - Kann **Parameter** entgegennehmen
        - Existiert kein eigener Konstruktor, erstellt der Compiler einen **Standardkonstruktor** (ohne Parameter)

        <br/>
        **Beispiel in C#:**
        ```csharp
        class Person
        {
            public string Name;
            public int Alter;

            // Konstruktor
            public Person(string name, int alter)
            {
                Name = name;
                Alter = alter;
            }
        }

        Person p = new Person("Max", 25); // Konstruktor wird aufgerufen
        ```
    </div>
