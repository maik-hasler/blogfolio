<Flashcard>
    <div slot="question">
        Was ist Methodenüberschreibung (Overriding)?
    </div>

    <div slot="answer">
        **Overriding** bedeutet, dass eine Unterklasse eine geerbte Methode der Oberklasse **neu implementiert** (überschreibt), wobei die **Methodensignatur gleich bleibt**.

        <br/>
        **Voraussetzungen in C#:**
        - Die Methode in der Basisklasse muss `virtual` oder `abstract` sein
        - Die Methode in der Unterklasse wird mit `override` markiert

        <br/>
        **Beispiel:**
        ```csharp
        class Tier
        {
            public virtual void LautGeben()
            {
                Console.WriteLine("...");
            }
        }

        class Katze : Tier
        {
            public override void LautGeben()
            {
                Console.WriteLine("Miau!");
            }
        }

        Tier t = new Katze();
        t.LautGeben(); // "Miau!" (dynamische Bindung)
        ```

        <br/>
        **Overriding vs. Overloading:**
        - **Overriding:** Gleiche Signatur, andere Implementierung in der Unterklasse
        - **Overloading:** Gleicher Name, **verschiedene** Parameter in derselben Klasse
    </div>
