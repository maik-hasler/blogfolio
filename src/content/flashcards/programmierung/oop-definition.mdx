    <div slot="question">
        Was ist objektorientierte Programmierung (OOP)?
    </div>

    <div slot="answer">
        **Objektorientierte Programmierung (OOP)** ist ein Programmierparadigma, bei dem Software als Zusammenspiel von **Objekten** modelliert wird, die Daten (Attribute) und Verhalten (Methoden) bündeln.

        <br/>
        **Die vier Grundprinzipien der OOP:**
        1. **Abstraktion:** Nur relevante Eigenschaften modellieren, Details verbergen
        2. **Kapselung:** Daten und Methoden zusammenfassen und den Zugriff kontrollieren
        3. **Vererbung:** Eigenschaften und Verhalten von einer Klasse an eine andere weitergeben
        4. **Polymorphie:** Objekte können in verschiedenen Formen auftreten

        <br/>
        **Vorteile:**
        - Näher an der realen Welt (Modellierung von Gegenständen/Konzepten)
        - Bessere Wiederverwendbarkeit durch Vererbung
        - Wartbarer durch Kapselung
        - Erweiterbar durch Polymorphie

        <br/>
        **Andere Paradigmen:** Prozedural, funktional, deklarativ
    </div>
</Flashcard>
