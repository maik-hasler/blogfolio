  <div slot="question">
    Was versteht man unter einer Funktion höherer Ordnung (Higher-Order Function)?
  </div>

  <div slot="answer">
      Eine Funktion höherer Ordnung ist eine Funktion, die andere Funktionen als Argumente erhält und/oder Funktionen als Ergebnis zurückgibt.
      
      <br/>
      Die Methode `Where` in C#/LINQ ist ein gutes Beispiel für eine Funktion höherer Ordnung.
      ```csharp
      public static IEnumerable<TSource> Where<TSource>(
        this IEnumerable<TSource> source,
        Func<TSource, bool> predicate);
      ```
      Der Parameter `predicate` ist eine Funktion vom Typ `Func<TSource, bool>`. Dadurch kann der Methode Funktionalität übergeben werden, die festlegt, welche Einträge im Ergebnis enthalten und welche ausgeschlossen werden sollen.
      ```csharp
      Func<int, bool> isDivisible(int n)
      {
          return x => x % n == 0;
      }

      var numbers = new List<int>() { 1, 2, 3, 4, 5, 6, 7, 8, 9 };
      Console.WriteLine(numbers.Where(isDivisible(3)));
      // => [3, 6, 9]
      ```
  </div>
